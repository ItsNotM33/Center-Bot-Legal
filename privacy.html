Politique de Confidentialité (Privacy Policy)

Dernière mise à jour : 22 février 2026

1. Données collectées

Le Bot peut traiter uniquement les données nécessaires à son fonctionnement :

  • ID utilisateur, ID serveur, ID salon
  • Actions de modération (ban, timeout, suppression de messages)
  • Paramètres de configuration du serveur (ex : salon Center/logs)

Le Bot ne collecte pas de mots de passe, de messages privés, ni de données sensibles volontairement.

2. Utilisation des données

Les données sont utilisées uniquement pour exécuter les commandes, produire des logs, et assurer le bon fonctionnement du Bot.

3. Partage des données

Les données ne sont ni vendues ni partagées à des tiers.

4. Conservation

Les données sont conservées uniquement le temps nécessaire au fonctionnement du Bot. Certaines configurations peuvent être conservées tant que le Bot est présent sur le serveur.

5. Sécurité

Nous appliquons des mesures raisonnables pour limiter l’accès non autorisé aux données, dans la limite des moyens disponibles sur l’hébergement.

6. Suppression des données

Vous pouvez demander la suppression des données liées à votre serveur (ex : configuration) en contactant le développeur via Discord.

7. Contact

Pour toute question relative à cette politique, contactez le développeur via Discord.
