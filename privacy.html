# Politique de Confidentialité (Privacy Policy) **Dernière mise à jour : 22 février 2026** ## 1. Données collectées Le Bot peut traiter les données suivantes, uniquement pour fonctionner : - Identifiants Discord (ID utilisateur, ID serveur, ID salon) - Données liées aux actions de modération (ex : bannissement, timeout, suppression de messages) - Paramètres de configuration du serveur (ex : salon Center/logs) Le Bot ne collecte pas : - de mots de passe ; - de messages privés ; - d’informations sensibles volontairement. ## 2. Utilisation des données Les données sont utilisées uniquement pour : - exécuter les commandes du Bot ; - afficher et centraliser des logs de modération dans le salon configuré ; - assurer la stabilité et la sécurité du service. ## 3. Partage des données Les données ne sont ni vendues, ni partagées avec des tiers. ## 4. Conservation Les données sont conservées uniquement le temps nécessaire au fonctionnement du Bot. Certaines configurations peuvent être conservées tant que le Bot est présent sur le serveur. ## 5. Sécurité Nous appliquons des mesures raisonnables pour éviter l’accès non autorisé aux données, dans la limite des moyens disponibles sur l’hébergement. ## 6. Suppression des données Vous pouvez demander la suppression des données liées à votre serveur (ex : configuration) en contactant le développeur via Discord. ## 7. Contact Pour toute question relative à cette Politique de Confidentialité, contactez le développeur via Discord.
